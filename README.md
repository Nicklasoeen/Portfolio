# Portfolio 1 — Nicklas Øen

![image](assets/images/hero-gradient.jpg)

A personal portfolio website showcasing selected front-end development projects, built as part of the Noroff curriculum.

## Description

This is a single-page portfolio site designed to present my work, skills, and background as a front-end development student. It features a modern, dark-themed design with smooth interactions and a modular JavaScript structure.

- Hero section with background image and overlay
- About section with CV and cover letter downloads
- Accordion-based fields/services overview
- Project showcase with interactive logo hover animations
- Responsive layout with a hamburger menu for mobile
- Footer with social links

## Built With

- HTML
- CSS
- JavaScript (ES Modules)
- [Google Fonts (Inter Tight)](https://fonts.google.com/specimen/Inter+Tight)

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:nicklasoeen/portfolio-1.git
```

2. Open the project folder:

```bash
cd portfolio-1
```

No dependencies to install — this is a vanilla HTML/CSS/JS project.

### Running

Open `index.html` directly in a browser, or use a local development server such as the VS Code Live Server extension.
